Day 1 Codeworks

  1. GIT:

    A free and open-source source control system designed to handle projects from small to large.

  2. Node:

    An open-source Javascript runtime environment for executing JS code server-side.

  3. NPM - Node Package Manager:

    An online repository for publishing of open-source Node.js projects

  4. VSCode:

    An IDE that is a "lite" version of Visual Studio. It is the environment where coding happens.
